RunSelendion

Setup: reset the button id count.

There is an optional second parameter: IMMEDIATE, CLICKABLE or LINK. IMMEDIATE means the script will be included directly in the results, CLICKABLE means it will be included but hidden, and you have to click a button to see the test, and LINK means the link to the test will be included. The default value is CLICKABLE.

Literal instead of contant

If a literal string (eg 'LINK') is used instead of a constant (eg LINK), there should be an error message.
