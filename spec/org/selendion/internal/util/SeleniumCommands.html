Selenium/HtmlUnit Commands Test

Selendion takes files generated from the Selenium IDE, and sends them to whatever browser you choose, using either Selenium or HtmlUnit. All standard Selenium IDE commands can be suffixed with IfAvailable. If this is present, then the command will not fail if the element is not found or another error occurs. If an error occurs, the command will pass but nothing will happen.

This allows you to write code like this

store loggedInUsername not found
storeText xPath=/html/body/p/span[3] loggedInUsername
