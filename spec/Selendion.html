What is Selendion?

Selendion is an extension of Concordion.

All concordion tests should run in selendion. The concordion test suite is part of the selendion build.

Commands

Concordion Commands

  • run
  • execute
  • set
  • assertTrue
  • assertFalse
  • verifyRows
  • echo

Concordion Commands that are extended in Selendion

  • assertEquals in Concordion
  • assertEquals in Selendion

Selendion Commands

  • Selenium commands
    • startBrowser
    • runSelenium
    • stopBrowser
    • switchJavaScript
  • Suite commands
    • addToSuite
    • runSuite
    • clearSuite
  • forEach
  • runSelendion
