RunSelendion

Setup: reset the button id count.

Run Selendion runs a selendion script when it is called. If the script has errors or exceptions, it will appear as a failed test. If it pass, hooray!. If it doesn't exist, an error message will be shown. If it exists but has no matching java file, it will be run as a basic selendion test. If it is not expected to pass and doesn't, the test will pass and the state result will be shown.
